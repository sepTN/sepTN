### 言語技術と日本語教育

- **Jepang.org**: JLPT、漢字、文法、語彙、アニメ情報、関連ツールを扱う、インドネシア語話者向けの日本語学習プラットフォーム
- **Yomi**: 日本語から英語の表現や漢字を検索できるリファレンスサイト [yomi.septn.com](https://yomi.septn.com)
- **Anime.Jepang.org**: インドネシア向けのアニメ番組表・データベース
- **Kanji.Jepang.org**: Jepang.orgのエコシステムの一部でありながら、独立して機能する漢字リファレンスツール
- 主な関心領域: **日本語**, **JLPT**, **漢字**, **データセット**, **検索**, **NLP（自然言語処理）関連のツール**

## 主なリポジトリ

- **kanji-data**: Node.js 向けのオフラインファーストな漢字データベース。ビルド時のシャーディングに対応  
  [github.com/sepTN/kanji-data](https://github.com/sepTN/kanji-data)
- **Bunpou Lens**: Kuromoji を使用した、クライアントサイドで動作する日本語構文解析ツール  
  [github.com/sepTN/bunpou-lens](https://github.com/sepTN/bunpou-lens)
- **kotowaza**: 日本語のことわざデータセット。インドネシア語と英語の訳付き  
  [github.com/sepTN/kotowaza](https://github.com/sepTN/kotowaza)
- **kanji-png-gif**: 漢字の筆順PNG・GIF画像をプログラムで生成するツール  
  [github.com/sepTN/kanji-png-gif](https://github.com/sepTN/kanji-png-gif)
